OVERVIEW

HTML stands for Hyper Text Markup Language. It is the standard markup language for creating Web pages which describes the structure of a Web page. It consists of a series of elements that tell the browser how to display the content.

OVERVIEW

CSS stands for Cascading Style Sheets. It describes how HTML elements are to be displayed on screen, paper, or in other media which saves a lot of work. It can control the layout of multiple web pages all at once. External stylesheets are stored in CSS files.
